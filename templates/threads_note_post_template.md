# Threads -> note 投稿テンプレート

## 使い方

`{{ }}` を今日のテーマに合わせて埋めます。最後に `prompts/threads_post_generator.md` で、200〜350文字程度の完成形に整えます。

## Template

```md
{{共感フック}}

{{自分の失敗談}}

{{今日の学び}}

{{読者への置き換え}}

{{noteへの自然なCTA}}
```

## Fill-in guide

### 共感フック

読者が抱えている迷いを、最初の1文にします。

例:

`AIを使いたいのに、何を頼めばいいか分からない。`

### 自分の失敗談

自分が実際につまずいたことを短く出します。

例:

`私も最初は、AIに全部任せようとして、かえって修正が増えました。`

### 今日の学び

失敗から分かったことを1つだけ書きます。

例:

`AIに渡す前に、目的と読者を1行で決めるだけで精度が変わります。`

### 読者への置き換え

読者が今日試せる形にします。

例:

`資料作りでも投稿文でも、最初の1行を決めてからAIに渡すと迷いが減ります。`

### noteへの自然なCTA

noteで読める続きを具体的に伝えます。

例:

`私が使っている入力前チェックはnoteに整理しました。プロフィールから読めます。`

## Review checklist

- 冒頭が読者の悩みになっている。
- 自分の失敗や実務感がある。
- 学びが1つに絞られている。
- 読者が今日試せる。
- CTAが売り込みすぎていない。
- noteで読める内容とCTAが一致している。
